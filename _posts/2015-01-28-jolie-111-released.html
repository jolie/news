--- layout: post title: Jolie 1.1.1 released date: '2015-01-28T12:36:00.000+01:00' author: Fabrizio Montesi tags: - release - jolie modified_time: '2015-01-28T12:36:51.425+01:00' blogger_id: tag:blogger.com,1999:blog-3824614534367079729.post-7563306591972479238 blogger_orig_url: https://jolie-lang.blogspot.com/2015/01/jolie-111-released.html --- Jolie 1.1.1 has been released.

Changelog


  • HTTP Compression Negotation is now supported and enabled by default.
  • The Jolie launcher script now uses the default Java parameters for RAM usage.
  • joliedoc now produces anchor links in documentation files correctly.
  • Bug fixes to the installer, which should now work better on MacOS and Windows.
